# Azure Cheatsheet

## Installation

### On Windows and verify

- [64 bit version](https://aka.ms/installazurecliwindowsx64)
- C:\az --version

### On Ubuntu

`$ curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash`

### Update the CLI

`$az upgrade`
